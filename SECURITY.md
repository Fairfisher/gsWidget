# Security Policy

## Supported Versions

We currently provide security updates for the following versions of **gsWidget**:


| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you believe you have found a security vulnerability, please report it by using one of the following methods:

1. **Private Vulnerability Reporting:** Use the [GitHub Private Vulnerability Reporting](https://github.com) feature in this repository. This is the preferred way to keep the conversation secure and private.
2. **Email:** Alternatively, you can send an email to **[DEINE-EMAIL-ADRESSE]**.

### What to include
To help us triage and fix the issue faster, please include:
- A description of the vulnerability and its potential impact.
- Step-by-step instructions to reproduce the issue.
- Any proof-of-concept code or screenshots if available.

### Our Commitment
We will acknowledge receipt of your report within 48 hours and aim to provide a more detailed response within 5 business days, including next steps for a fix. We follow the principle of **Coordinated Vulnerability Disclosure** and ask that you give us reasonable time to fix the issue before sharing any information publicly.
